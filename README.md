# BrandPulse — Campaign Marketing Platform

Turn marketing chaos into measurable growth.

BrandPulse gives marketers a single command center to launch, track,
and optimize campaigns across every channel, powered by AI that thinks
like a senior strategist.

## Why it matters
- Save 10+ hours/week — no more jumping between tools
- 3x faster campaign launches with AI-generated briefs and copy
- Catch underperforming ads early with real-time alerts
- Make smarter budget decisions with unified analytics

## What it does
- Live dashboard with reach, CTR, ROAS and conversion metrics
- Campaign manager — create, filter, and track progress in one place
- AI Advisor — generates ad copy, strategies, personas and A/B test plans
- Funnel analytics with audience breakdown by age and channel

## Quick start
1. Open index.html in your browser
2. Go to Settings and paste your Anthropic API key
3. Start a new campaign or ask the AI Advisor anything

## Built with
HTML, CSS, JavaScript, Anthropic Claude API
